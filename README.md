# Global Scheduling Application

## Overview
A Java desktop application for managing global travel appointments. The application includes multilingual support, time zone conversion, multi-currency display, and Docker containerization.

## Features
- Manage customer appointments
- Time zone conversion (UTC, ET, MT)
- Multi-language support
- Multi-currency price display
- Docker support

## Technologies
- Java
- JavaFX
- Maven
- Docker
- Git
- GitHub
  
## Skills Demonstrated
- Object-Oriented Programming
- Desktop Application Development
- Internationalization
- Time Zone Handling
- Version Control
- Containerization with Docker
